Scribble

Single character handwriting recognition.


For use in Nokia programs
and for external licensing.

Products

Scribble engine

Java engine that recognises strokes on device, integrated with Z Launcher and demonstration Scribble test application.

Scribble test app

Scribble engine powered demonstration and character capture application.


Scribble tools

Lightweight javascript tools for processing collected user stroke data for refining recognition models and test data sets.

How it works

1. Write

Java handwriting recognition engine easily integrated with estate challenged applications.

2. Analyze

Signature characteristics (eg., loops, cusps, etc) of individual glyphs are determined using feature extraction.


3. Compare

Characteristics are compared against local collection of known glyphs (the model) for similarities until a match is found.

4. Respond

Result of comparison is returned to the application.

Continuous improvement

1. Harvest

Instrumentation integrated with recognition engine collects and stores examples of user entered characters.

2. Classify

Collected data is manually rendered and classified using unique web-based tooling designed specifically for purpose.


3. Integrate

Classified data is merged with existing test data to create progressively more stringent tests for the model.

4. Refine

Model (see 3 above) quality is continuously improved through the inclusion of enriched test data.
